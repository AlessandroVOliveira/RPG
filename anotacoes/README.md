# Anotações de aventura

Pasta para anotações soltas de sessão em `.txt` (ou `.md`, se preferir formatação) — recapitulativos, ganchos de história, NPCs, decisões da mesa, o que quiser registrar entre uma sessão e outra.

Sem estrutura obrigatória. Sugestão de nome de arquivo: `AAAA-MM-DD sessao.txt`.
