# Memórias de regras

Resumos de regras extraídos dos PDFs em `manuais-t20/` e `manuais-classico/`, organizados por sistema — ler estes arquivos é rápido e evita reabrir PDFs enormes toda vez que uma regra precisa ser conferida.

- **[t20/](t20/README.md)** — Tormenta20 ("Jogo do Ano"). Usado pelo Andor **arquivado** em `personagens/andor-t20/`. Não está mais em evolução.
- **[classico/](classico/README.md)** — Tormenta RPG (edição clássica, pré-T20). Usado pelo Andor **ativo** em `personagens/andor-classico/`. É onde novas regras devem ser extraídas e documentadas a partir de agora.

Se algo não estiver documentado no tema certo, consulte o PDF correspondente em `manuais-t20/` ou `manuais-classico/` e depois complete o `.md` — ver `CLAUDE.md` na raiz para o fluxo completo.
