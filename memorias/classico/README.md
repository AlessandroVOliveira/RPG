# Memórias de regras — Tormenta RPG (edição clássica)

Resumo das regras da edição **clássica** do Tormenta RPG (pré-T20 — usa CA, Bônus Base de Ataque, classes com níveis de conjurador, Talentos ao invés de Poderes), extraídas dos PDFs em `manuais-classico/`. Esta é a edição que o mestre vai usar a partir de agora; o Andor em `personagens/andor-classico/` é construído com base nestas regras.

## Arquivos já extraídos

- **[regras-base.md](regras-base.md)** — geração de atributos, tabela de modificadores, perícias (fórmula de teste, graduações, tabela de habilidade-chave/treinada/penalidade de armadura).
- **[raca-humano.md](raca-humano.md)** — traços raciais do Humano (+2 em duas habilidades, 2 talentos extras, 2 perícias treinadas extras).
- **[guerreiro.md](guerreiro.md)** — características de classe, BBA cheio, Técnica de Luta (talentos de combate por nível).
- **[talentos.md](talentos.md)** — talentos de combate relevantes para build de tanque com escudo (livro básico + Compilado de Talentos: Esquiva, Ataque com Escudo Aprimorado/Escudo Fraterno, Especialização em Armadura, Foco/Especialização em Arma, Duro de Matar/Duro de Ferir, Reflexos de Combate etc.), os 3 Talentos de Destino confirmados (Fortitude Maior/Vontade de Ferro/Reflexos Rápidos, +2 cumulativo cada) e a variante de classe Estilo Colosso (Manual do Combate — avaliada e descartada para o Andor, ver arquivo).
- **[combate.md](combate.md)** — fórmulas de CA, PV, ataque, dano, testes de resistência, iniciativa, ações por rodada.
- **[equipamento.md](equipamento.md)** — tabela de armaduras/escudos completa, armas relevantes (espada longa, machado de batalha), dinheiro inicial.

Ver `CLAUDE.md` na raiz para o fluxo de consulta (memórias primeiro, PDF depois, sempre citando a página impressa).

## O que ainda NÃO foi extraído

Magias (Cap. 8), Talentos de Perícia/Magia/Poderes Concedidos/Tormenta/Racial/Regional/Épicos do Compilado (só a seção Combate e Destino foram lidas), outras raças e classes do Manual das Raças/Manual do Combate (Cavaleiro, Lutador, tribos de Bárbaro, Artes de Monge, Clãs de Samurai, Golpes Marciais/Posturas/Escolas de Combate/Gritos de Poder), classes de prestígio do Manual de Classes de Prestígio (nenhuma delas é boa opção pra um Humano Guerreiro tanque — são majoritariamente arcanas/furtivas), Cap. 6 do livro básico (Tendência/Deuses/Pontos de Ação), Cap. 10 (Mestre/Tesouros/Itens Mágicos), Bestiário. Foi extraído o necessário pra montar e progredir (1º-20º nível) um Humano Guerreiro tanque de espada e escudo.

## Livros disponíveis em `manuais-classico/` e o que cada um cobre

- **`tormenta-rpg-modulo-basico-biblioteca-elfica.pdf`** — livro básico: regras centrais, atributos, perícias, classes principais, combate, magia geral. Ponto de partida para (quase) tudo.
- **`tormenta-rpg---manual-das-racas.pdf`** — raças jogáveis e suas habilidades.
- **`tormenta-rpg---manual-do-combate.pdf`** — classes e regras marciais (ex.: Guerreiro, Talentos de combate).
- **`tormenta-rpg---manual-do-arcano.pdf`** — classes e magias arcanas.
- **`tormenta-rpg---manual-do-devoto.pdf`** — classes e magias divinas.
- **`tormenta-rpg---manual-do-malandro.pdf`** — classes furtivas/especialistas.
- **`tormenta-rpg-manual-de-classes-de-prestigio-biblioteca-elfica.pdf`** — classes de prestígio.
- **`tormenta-rpg-compilado-de-talentos-biblioteca-elfica.pdf`** — talentos gerais compilados.
- **`tormenta-rpg-guia-completo-e-definitivo-de-armas-armaduras-e-escudos-biblioteca-elfica.pdf`** — equipamento de combate.
- **`tormenta-rpg---bestiario-de-arton-vol.1.pdf`** / **`vol.2.pdf`** — monstros.
- **`tormenta-rpg-manual-de-criacao-de-monstros-biblioteca-elfica.pdf`** — regras para criar monstros (referência, não afeta PJ).
- **`tormenta-rpg---o-mundo-de-arton.pdf`** — livro de região/mundo (mesma edição desta pasta — atenção: uma cópia deste arquivo também existe em `manuais-t20/`, mas lá ela é a edição "errada" para o T20).
- **`tormenta-rpg-o-mundo-dos-deuses.pdf`**, **`tormenta-rpg---o-desafio-dos-deuses.pdf`** — divindades, Poderes Concedidos/clérigos.
- **`tormenta-rpg-elfos-negros-biblioteca-elfica.pdf`**, **`tormenta-rpg-grupo-do-mal-biblioteca-elfica.pdf`**, **`tormenta-rpg-novos-horizontes-biblioteca-elfica.pdf`**, **`tormenta-rpg-pistoleiro-arcano-biblioteca-elfica.pdf`** — suplementos temáticos/opcionais.
- **`tormenta-rpg---expedicao-a-alianca-negra.pdf`** — módulo de aventura (não é material de regras de personagem).
- **`tormenta-rpg-ficha-de-personagem-editavel-biblioteca-elfica.pdf`** — ficha em branco oficial desta edição.

## Como preencher isto

Siga o mesmo padrão de `memorias/t20/`: um arquivo por tema (raça, classe, talentos, equipamento etc.), cada um citando a página impressa do livro de origem. Atualize esta lista de arquivos conforme forem criados.
