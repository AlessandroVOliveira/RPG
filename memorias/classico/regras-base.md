# Regras base — Tormenta RPG (edição clássica)

Fonte: `manuais-classico/tormenta-rpg-modulo-basico-biblioteca-elfica.pdf` ("Módulo Básico"). Página impressa do livro entre parênteses (não é o número de página do PDF — neste arquivo, PDF e impressa coincidem 1:1 até pelo menos a p.300).

Sistema d20/OGL (base D&D 3.5): Bônus Base de Ataque (BBA), Classe de Armadura (CA), Testes de Resistência (Fortitude/Reflexos/Vontade), Talentos (feats) em vez de Poderes. Bem diferente de T20 — ver `memorias/t20/` para a versão anterior (não usar como referência de regra aqui).

## Habilidades (atributos) — Cap. 1, pág. 24-27

Seis habilidades: Força, Destreza, Constituição, Inteligência, Sabedoria, Carisma.

Tabela de modificador (pág. 26):

| Valor | Modificador |
|---|---|
| 1 | −5 |
| 2-3 | −4 |
| 4-5 | −3 |
| 6-7 | −2 |
| 8-9 | −1 |
| 10-11 | 0 |
| 12-13 | +1 |
| 14-15 | +2 |
| 16-17 | +3 |
| 18-19 | +4 |
| 20-21 | +5 |
| 22-23 | +6 |

**Geração de habilidades** — vários métodos possíveis (pág. 27): Padrão (4d6, descarta o menor, x6 — média 12-13), Clássica (3d6 x6 — média 10-11), Heroica (2d6+6 x6 — média 13), Audaciosa (1d20 x6), **Elite** (array fixo, sem rolar: 17, 15, 13, 12, 10, 8).

⚠️ O Andor usa os valores **rolados nos dados** (17, 15, 13, 12, 12, 11 — ordenados), carregados da versão T20. Não é o array Elite (que seria 17,15,13,12,10,8) — é uma rolagem própria feita pelo jogador, mantida entre as duas versões do personagem por pedido dele.

## Perícias — Cap. 4, pág. 76-89

Teste de perícia: **1d20 + graduações + modificador de habilidade + outros modificadores**.

**Graduações** (ranks):
- Perícia treinada: nível do personagem + 3.
- Perícia não treinada: metade do nível do personagem (arredonda pra baixo). Perícias marcadas "Treinada" na tabela abaixo só podem ser testadas se o personagem for treinado nelas (não existe teste "sem treinamento").

**Perícias treinadas por classe** (pág. 78): 2 + modificador de Inteligência para Clérigo/Feiticeiro/**Guerreiro**/Mago/Paladino (o grupo mais baixo da tabela). Escolhidas no 1º nível, não podem ser trocadas depois.

**Humanos ganham +2 perícias treinadas extras**, que podem ser escolhidas entre perícias de fora da lista da classe (pág. 78) — isso é *além* do bônus de raça já listado em `raca-humano.md`.

## Talento "Treino em Perícia"

Concede uma perícia treinada adicional, que pode ser de fora da lista da classe (pág. 78) — bom pra Guerreiro (poucas perícias de classe) se sobrar talento geral.
