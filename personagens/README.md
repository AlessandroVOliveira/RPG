# Personagens

Um personagem (Andor Duvane), duas versões — o mestre trocou de edição do sistema (Tormenta20 → Tormenta RPG clássico), então cada versão vive na sua própria pasta com ficha, dados e docs de build.

- **[andor-classico/](andor-classico/)** — versão **ativa**, construída com as regras em `memorias/classico/` / `manuais-classico/`. É aqui que o trabalho novo acontece.
- **[andor-t20/](andor-t20/)** — versão **arquivada** (congelada), construída com as regras em `memorias/t20/` / `manuais-t20/`. Mantida só como referência histórica.

Os atributos base (rolados nos dados: 17, 15, 13, 12, 12, 11) são os mesmos nas duas versões — o que muda é raça, classe, perícias, talentos/poderes e equipamento, adaptados a cada sistema.

`anotacoes/` e `Imagens/` na raiz do projeto são compartilhadas entre as duas versões (mesma campanha, mesmo personagem).
